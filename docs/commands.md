# Команды бота (меню через `/`)

## Базовые команды
- `/start` — главное меню и приветствие.
- `/help` — гайд и справка по боту.
- `/rules` — правила клана.
- `/register` — регистрация (в группе отправит ссылку в ЛС).
- `/profile` — мой профиль.
- `/stats` — моя статистика.
- `/targets` — меню целей на войне.
- `/complaint` — подать жалобу.
- `/notify` — настройки личных уведомлений (ЛС).

## Админские команды
- `/admin` — админ-панель.
- `/users` — список зарегистрированных пользователей.
- `/complaints` — список жалоб.
- `/missed` — кто не атаковал сейчас (КВ/ЛВК).
- `/set_chat_notify` — настройки уведомлений общего чата.
- `/update_commands` — обновить список команд.
